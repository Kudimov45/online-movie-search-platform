# online-movie-search-platform

Платформа для поиска фильмов, на которой пользователи могут искать фильмы и добовлять их в избранное

Целью данного приложение было создать логику обработки API и дать возможность пользователю не только искать фильмы, но и производить вход в свой аккаунт для сохранения фильмов в избранное
